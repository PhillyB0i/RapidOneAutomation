General:
Hotkey List: LEFT CONTROL + F1
Create Financial Document: LEFT CONTROL + F2
Jump to latest patient: LEFT CONTROL + F3
Switch schedule to today: LEFT CONTROL + F6

Reports:
Jump to Treatment Report: LEFT CONTROL + F7
Jump to Patient Report: LEFT CONTROL + F8
Jump to Attendance Managment: LEFT CONTROL + F9
Jump to Custom daily income report: LEFT CONTROL + F10

CRM:
Move to lead managment: LEFT ALT + F1
Move to next page: LEFT ALT + Right arrow
Move to previous page: LEFT ALT + Left arrow
Jump to first page: LEFT ALT + Up arrow
Jump to last page: LEFT ALT + Down arrow
Show latest leads for today: LEFT ALT + F2
Show latest leads for last 7 days: LEFT ALT + F3
Show latest leads for last 14 days: LEFT ALT + F5
